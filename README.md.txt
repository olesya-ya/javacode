# сравнительный анализ систем Яндекс.Геокодер, DaData, GigaChat на основании изучения API-контрактов и документации сервиса

### Цель
Gринятия решения об использовании сервиса прямого/обратного геокодирования.

### Методология
* **Набор данных:** 100 адресов и 100 координат (регионы РФ: Москва, СПб, Екатеринбург, Новосибирск и др.)
* **Сервисы:**
  * DaData — коммерческий геокодер с поддержкой КЛАДР/ФИАС
  * Яндекс.Геокодер — геосервис от Яндекса
  * GigaChat — LLM от Сбера
* **Метрики:**
  * Прямое геокодирование: среднее расстояние между координатами (в метрах)
  * Обратное геокодирование: доля корректных адресов по экспертной оценке (0-100%)


## Результаты

### 1. Обратное геокодирование (координаты в адрес)

| Сервис | % корректных адресов | Примечания |
| :--- | :--- | :--- |
| **DaData** | 84.7% | Возвращает полные адреса с индексом, улицей, домом. |
| **Яндекс.Геокодер** | 82.9% | Возвращает полные адреса с индексом, улицей, домом. |
| **GigaChat** | 0% | Выдаёт неверные адреса. |

**Пример:**
*   **DaData:** 354061, Краснодарский край, г Сочи, ул Советская, д 26 (полный адрес)
*   **Яндекс:** Россия, Краснодарский край, Сочи, микрорайон Центральный, Советская улица, 26,  354000 (полный адрес)
*   **GigaChat:** Россия, Ульяновск, улица Гончарова, дом 17 (примерный адрес, точный адрес может отличаться от фактического местоположения координат) (вымышленный адрес)

---

### 2. Прямое геокодирование (адрес в координаты)

| Сравнение | Среднее расстояние | Вывод |
| :--- | :--- | :--- |
| **Яндекс ↔ DaData** | 112.6 км | Сервисы иногда выдают разные координаты. |
| **Яндекс ↔ GigaChat** | 387.1 км | Непригоден. |

**Пример (Адрес: 109012, г. Москва, Красная площадь, д. 1):**
*   **Яндекс:** 55,755277, 37,61768 
*   **DaData:** 55,7552921, 37,6176294 (отклонение: **3.5 м**)
*   **GigaChat:** 55,75396, 37,62039 (отклонение: **223.3 м**)

---

## Вывод и рекомендации

*   ✅ **Рекомендуется: DaData или Яндекс.Геокодер**
    *   Наивысшая точность как для прямого, так и для обратного геокодирования.
    *   Полные структурированные данные: индекс, адрес, дом.
    *   Стабильный API и отличная поддержка российской адресной системы.

*   ❌ **Не рекомендуется: GigaChat**
    *   Не является геокодером — это LLM, не обученная точным геоданным.
    *   Ошибки достигают десятков километров.

**Итог:** Для production-систем, требующих высокой точности, оптимальный выбор — **DaData или Яндекс.Геокодер**.
